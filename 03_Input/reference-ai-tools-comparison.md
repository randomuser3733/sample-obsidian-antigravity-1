# AIツール比較リファレンス

| ツール | 強み | 弱み | 代表用途 |
|--------|------|------|----------|
| Cursor | IDE統合、コーディング支援 | 学習コストが高い | ポートフォリオ作成、学習 |
| Claude | 長文理解、説明が詳しい | コード実行はできない | 学習サポート、質問対応 |
| ChatGPT | 汎用性が高い | コード生成の精度 | 学習サポート、調べ物 |
| GitHub Copilot | VS Code統合 | 設定が複雑 | コーディング支援 |

## 選定観点
- 学習しやすさ: 初心者でも使えるか
- コスト: 無料プランがあるか
- サポート: コミュニティが充実しているか
