# Cursor基礎

## 概要
Cursor基礎に関する知見を整理する。CursorはAIペアプログラミングを活かした高速開発が強み。初心者でも使いやすいが、基礎機能を体系的に押さえると応用も楽になる。

## 重要ポイント
- チャットペインで質問すると、コードの提案が返ってくる
- エラーが出た時に、エラーメッセージを貼り付けると解決方法を教えてくれる
- プロジェクトごとにモデル設定を変え、ClaudeとGPTを使い分ける
- ⌘Pでファイル移動、⇧⌘Kでコマンド検索などよく使う操作を覚える
- マルチファイル編集機能で、複数のファイルを同時に編集できる
